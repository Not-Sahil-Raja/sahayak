## Get Started

1. Clone the repository

```bash
git clone
```

2. Install dependencies

```bash
npm install
```

3. Create a `.env` file in the root directory and add the following

```bash
NEXT_PUBLIC_API_URL="your api url"
```

4. Run the project

```bash
npm run dev
```

5. Open the browser and go to `http://localhost:3000`
